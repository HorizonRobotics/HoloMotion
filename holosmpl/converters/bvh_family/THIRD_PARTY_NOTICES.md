# HoloSMPL Third-Party Notices

HoloSMPL contains adapters for converting public human motion datasets into the
project canonical SMPL-like representation.

## LaFAN1 BVH Adapter

The LaFAN1 BVH adapter includes code adapted from the LaFAN-to-SMPLX reference
logic. Its MIT license is retained at:

```text
holosmpl/converters/bvh_family/LICENSE_lafan_to_smplx.txt
```

No dataset files, SMPL/SMPL-X model files, or generated HDF5/NPZ artifacts are
included in this package.
